## v4 — Confidence 门槛 + 偏空消灭 + mild_bull 收窄 + 趋势疲劳优化 (2026-04-24)

**触发**: 135 条信号（129 条 5d 回填）深度分析（Polly + 小龙虾 独立分析 + 交叉验证）

### 关键发现（Polly + 🦞 交叉验证）

**1. Confidence 是被低估的金矿** 🦞

| 区间 | 样本 | 5d 胜率 |
|------|------|---------|
| 高(70+) | 68 | **72.1%** ✅ |
| 中(40-69) | 38 | 50.0% |
| 低(<40) | 23 | **47.8%** ❌ ≈ 抛硬币 |

差异 72.1% vs 47.8% — 最容易拿到的 alpha。

**2. 偏空区间是个坑**

| 方向 | 样本 | 5d 胜率 | 平均 5d 收益 |
|------|------|---------|-------------|
| 看空 (≤-25) | 73 | **68.5%** ✅ | -2.17% |
| 看多 (≥50) | 18 | **66.7%** ✅ | +3.46% |
| 中性 (-8~8) | 11 | 63.6% | +3.70% |
| 偏多 (8~50, v3→中性) | 17 | **35.3%** ❌ | -1.14% |
| **偏空 (-25~-8)** | **10** | **40.0%** ❌ | **+2.07%** ← 反向 |

偏空 40% 胜率 + 正向收益，信号完全反向。

**3. 体制×方向交叉**

| 体制×方向 | 样本 | 5d 胜率 |
|-----------|------|---------|
| bear + 看空 | 31 | **74.2%** ✅ 最佳组合 |
| mild_bull + 看空 | 26 | 61.5% |
| mild_bull + 看多 | 15 | 60.0% |
| mild_bull + 中性观望 | 13 | 53.8% |
| mild_bull + 偏多 | 7 | 57.1% |
| **mild_bull + 偏空** | **9** | **33.3%** ❌ |

**4. 连续同向信号分方向分析** 🦞

| 方向 | 连续样本 | 5d 胜率 |
|------|---------|---------|
| 看空 | 61 | **68.9%** ✅ 不衰减 |
| 看多 | 12 | 66.7% |
| 中性观望 | 7 | 57.1% |
| 偏空 | 4 | **25.0%** ❌ |

看空连续信号 68.9% 仍然靠谱，不应衰减。002111 问题不是"连续"，是 confidence 中等（50-74）。

**5. 002111 威海广泰 pattern**

4/13~4/17 连续 5 天看多，confidence 50-74（中等区间），5d 全部 wrong。
真因不是"连续看多"，而是"confidence 不够高却 score 很高"——中等 confidence 本身就不靠谱（50.0%）。

### 改动清单

| 优先级 | 改动 | 文件 | 依据 |
|--------|------|------|------|
| **P0** | conf < 40 → 强制中性观望 | server.py | 低 conf 47.8% vs 高 conf 72.1% |
| **P1** | 删除"偏空"方向，[-25,-8) 全归中性 | server.py (×2) | 40% 胜率 + 反向收益 +2.07% |
| **P2** | mild_bull 下看空阈值 -25 → -35 | server.py (×2) | mild_bull + 偏空 33.3% |
| **P3** | 趋势疲劳仅对看多触发，看空免疫 | guardrails.py | 看空连续 68.9% 不应衰减 |

### 新信号阈值 (v4)

| 评分 | 信号 | 备注 |
|------|------|------|
| conf < 40 | 中性观望 | P0 门槛，优先于所有评分判断 |
| ≥ +50 | 看多 | mild_bear 下降为中性 |
| +8 ~ +50 | 中性观望 | v3 消灭偏多 |
| -8 ~ +8 | 中性观望 | |
| -25 ~ -8 | 中性观望 | P1 消灭偏空 |
| ≤ -25 | 看空 | mild_bull 下收窄到 ≤-35 (P2) |

### 置信度计算 (v4)

```
base = min(|adjusted_score|, 100)
if |score| >= 85: base × 0.7  （过度自信衰减）
if 方向翻转:     base × 0.6  （翻转不靠谱）
if conf < 40:    → 强制中性  （P0 门槛）
```

### 小龙虾 Review 要点

> "Confidence 是个被低估的金矿。高 confidence 72.1%，中低 confidence 基本是抛硬币。v4 应该引入 confidence 门槛——低于 40 的信号直接降级为中性，不管 score 多极端。"

> "连续同向的数据比想象的复杂。看空连续 68.9%（61条）— 没衰减，说明看空连续加仓是对的。连续偏空 25%（4条）— 烂到底了。'连续同向额外衰减'要分方向——不能一刀切。"

> "002111 的真正 pattern: 两个问题——极端看空到底后翻多时识别不够快，看多 confidence 只有 50-74，中等 confidence 本身就不靠谱。"

### 未执行（持续追踪）

- conf 40~69 区间是否需要进一步衰减（当前 50.0% 刚好边界）
- mild_bull -35 阈值是否需要根据更多数据微调
- 个股 confidence 校准（不同股票的 confidence 可能需要个性化调整）
- v3 偏多消灭验证——17 条 35.3%，与 v3 前的 27.3% 接近，符合预期
